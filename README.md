### Informatik-projekt CO2 Måler

# Problemformulering

Hvordan kan vi være med til at sænke CO2 niveauerne i vores klasse? Vi regner med, at lave en CO2-måler der kan fortælle (via en buzzlyd), når klasseværelsets CO2-niveau er for højt. Derudover vil måleren også vise vha. farver hvor høj CO2 niveauet er pt. i klassen.
